# ddu-commands.vim

Commands for ddu.vim


## Required

### denops.vim

https://github.com/vim-denops/denops.vim


### ddu.vim

https://github.com/Shougo/ddu.vim


## Configuration

```vim
" Use file_rec source.
Ddu file_rec

" Change input option
Ddu file -input=foo

" Use ui param
Ddu file -ui-param-ff-startFilter=v:true
```
